# Действия в консоли

### показать(строка="")
### print(string="")

##### Аргументы
`string/строка` - строка, значение по умолчанию - ""

##### Действие
Выводит строку `string/строка` в консоль

##### Возвращает
None

---

### ввести(строка="")
### input(string="")

##### Аргументы
`string/строка` - строка, значение по умолчанию - ""

##### Действие
Выводит необязательную строку `string/строка`, затем ожидает ввода от пользователя

##### Возвращает
Строка

---

# Действия над списками

### append|добавить(list, value)

##### Аргументы

- **_list_** - список
- **_value_** - любое значение

##### Действие

Добавляет значение **_value_** в конец списка **_list_**

##### Возвращает

None

### pop|получить*и*удалить(list ~~, index=-1~~)

##### Аргументы

- **_list_** - список
- ~~**_index_** - число

##### Действие

Удаляет из списка **_list_** ~~значение по индексу **_index_** или~~ ~~, если не указан~~ последнее значение

##### Возвращает

Удалённое значение

### extend|расширить(list1, list2)

##### Аргументы

- **_list1_** - список
- **_list2_** - список

##### Действие

Значения из списка **_list2_** добавляются в конец списка **_list1_**

### length|длина(list)

##### Аргументы

- **_list_** - список

##### Действие

Находит количество элементов в списке **_list_**

##### Возвращает

Целое число

### join|соединить(separator, string_list)

##### Аргументы

- **_separator_** - строка
- **_string_list_** - список строк

##### Действие

Объединяет ~~любые элементы~~ строки списка **_string_list_** в строку, разделяя их строкой **_separator_**

### Возвращает

Строка

### map|исполнить*для*каждого(function, list)

##### Аргументы

- **_function_** - функция
- **_list_** - список

##### Действие

Применяет функцию **_function_** ко всем элементам списка **_list_** и возвращает результаты исполнения функции в виде списка

##### Возвращает

Список

### insert|вставить(list, index, value)

##### Аргументы

- **_list_** - список
- **_index_** - число
- **_value_** - любое значение

##### Действие

Вставляет значение **_value_** в список **_list_** на позицию **_index_**, сдвигая все последующие элементы

##### Возвращает

None

### change|изменить|mutate|мутировать(list, index, value)

##### Аргументы

- **_list_** - список
- **_index_** - число
- **_value_** - любое значение

##### Действие

Вставляет значение **_value_** в список **_list_** на позицию **_index_**, заменяя предыдущее значение

##### Возвращает

None

### reverse|обратить(list)

##### Аргументы

- **_list_** - список

##### Действие

Разворачивает список **_list_**

##### Возвращает

None

### reversed|обратный(list)
